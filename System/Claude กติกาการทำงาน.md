# ⚙️ กติกาการทำงานของ Claude (อ่านทุกรอบก่อนเริ่มงาน)

เอกสารนี้คือ SOP สำหรับ Claude ทุก session ที่เข้ามาประมวลผล TKG Vault — ทำตามลำดับทุกข้อ

## รอบทำงานมาตรฐาน

1. **อ่าน Inbox ทั้งหมด** — หาโน้ตที่ `สถานะ: รอทำ`
2. **ทำทีละงาน:**
   - เปลี่ยนสถานะเป็น `กำลังทำ`
   - เลือก skill ตามตารางด้านล่าง แล้วทำงานตามข้อมูลที่เจ้าของกรอก ถ้าข้อมูลไม่พอ ให้ทำเวอร์ชันที่ดีที่สุดจากที่มี แล้วระบุคำถามไว้ท้ายผลงาน
   - เขียนผลงานเป็นโน้ตใหม่ใน Outputs หมวดที่ถูกต้อง ชื่อไฟล์ `YYYY-MM-DD — ชื่องาน.md`
   - โครงโน้ตผลงาน: frontmatter (ประเภท, สถานะ: เสร็จ — รอตรวจ, วันที่เสร็จ, ช่องทาง, งานต้นทาง) + เนื้องานพร้อมใช้ + เช็คลิสต์ + กล่องสั่งแก้ (ดูตัวอย่างที่ [[Outputs/Content/2026-07-02 — ตัวอย่างโพสต์ให้ความรู้]])
   - กลับไปเปลี่ยนสถานะโน้ตต้นทางใน Inbox เป็น `เสร็จ — รอตรวจ` + ใส่ลิงก์ไปผลงาน
3. **เช็คงานสั่งแก้** — หาโน้ตใน Outputs ที่ `สถานะ: สั่งแก้` อ่านคอมเมนต์ท้ายโน้ต แก้งานในโน้ตเดิม แล้วเปลี่ยนสถานะกลับเป็น `เสร็จ — รอตรวจ`
4. **อัปเดต [[Home]]** — ตารางงานใน Inbox, ผลงานล่าสุด (5 รายการ), สถิติเดือนนี้, เวลารอบล่าสุด
5. **Archive** — งานใน Inbox ที่เสร็จ/อนุมัติเกิน 30 วัน ย้ายไป `Archive/`

## หลักการเขียนงาน

- ภาษาไทย โทนมืออาชีพ เห็นอกเห็นใจลูกค้า สร้างความมั่นใจ — ตาม persona ของ TKG Hair Center
- เนื้องานต้อง "พร้อมใช้" — เจ้าของ copy ไปใช้ได้ทันทีโดยไม่ต้องแก้
- ภาพ Before-After แนะนำให้ใช้ภาพลูกค้าจริงที่ได้รับอนุญาตเสมอ ไม่แนะนำภาพ AI แทนผลงานจริง
- ห้ามลบโน้ตของเจ้าของ ย้ายได้อย่างเดียว (ไป Archive)
