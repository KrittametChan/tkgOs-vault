# 📖 คู่มือใช้งาน TKG Vault

## ติดตั้งครั้งแรก (ทำครั้งเดียว)

1. ดาวน์โหลด Obsidian (ฟรี) จาก obsidian.md ติดตั้งบนคอม
2. เปิด Obsidian → "Open folder as vault" → เลือกโฟลเดอร์ `TKG Vault` นี้
3. ติดตั้ง Obsidian บนมือถือ (App Store / Play Store)
4. ตั้ง sync: ใช้ iCloud Drive (ฟรี — ต้องย้าย vault ไปไว้ในโฟลเดอร์ iCloud แล้วแจ้ง Claude ให้เพิ่มโฟลเดอร์ใหม่) หรือ Obsidian Sync (~$4/เดือน ไม่ต้องย้ายโฟลเดอร์)
5. (แนะนำ) Settings → Appearance → Themes → ติดตั้งธีม "Minimal" แล้วเปิด dark mode
6. (แนะนำ) Settings → Appearance → CSS snippets → เปิด `tkg-brand` เพื่อใช้สีแบรนด์ทอง

## วิธีสั่งงาน (แบบปุ่มกด — แนะนำ)

ต้องติดตั้ง plugin ก่อน (ดูหัวข้อถัดไป — ทำครั้งเดียว)

1. เปิด [[Home]] → กดปุ่มประเภทงานที่ต้องการ
2. ระบบเด้งถามชื่องาน เช่น "โพสต์โปรฯ สิงหา" → Enter
3. โน้ตถูกสร้างใน `Inbox` พร้อมฟอร์มและวันที่อัตโนมัติ → กรอกรายละเอียด
4. เสร็จแล้วรอได้เลย — Claude เช็ค Inbox ทุกเช้า 9:00 (คอมต้องเปิดและแอป Claude ทำงานอยู่)
5. อยากให้ทำทันที: เปิดแอป Claude แล้วพิมพ์ "เช็ค Inbox ใน TKG Vault เดี๋ยวนี้"

*วิธีสำรอง (ไม่มี plugin):* เปิด template → คัดลอกเนื้อหาไปสร้างโน้ตใหม่ใน `Inbox` แล้วกรอกเอง

## ติดตั้ง plugin สำหรับปุ่มกด (ครั้งเดียว)

1. Settings → Community plugins → Turn on community plugins
2. Browse → ติดตั้งและ Enable: **Buttons** และ **QuickAdd**
3. ปิด-เปิด Obsidian ใหม่ 1 ครั้ง → ปุ่มบน Home พร้อมใช้ (การตั้งค่า QuickAdd ทั้ง 9 ปุ่มถูกเตรียมไว้ให้แล้ว)
4. บนมือถือ: plugin และการตั้งค่า sync มากับ vault อัตโนมัติ ใช้ได้เลย

ถ้ากดปุ่มแล้วขึ้น "command not found": เปิด Settings → QuickAdd → ดูว่ามี choice 9 อัน และไอคอนสายฟ้า (⚡) ของแต่ละอันเปิดอยู่

## วิธีตรวจงานและสั่งแก้

- ผลงานอยู่ใน `Outputs/` แยกหมวด — สถานะ "เสร็จ — รอตรวจ"
- พอใจแล้ว: เปลี่ยนสถานะใน frontmatter เป็น `อนุมัติแล้ว` (หรือปล่อยไว้ก็ได้)
- อยากแก้: พิมพ์สิ่งที่ต้องการต่อท้ายโน้ตผลงาน แล้วเปลี่ยนสถานะเป็น `สั่งแก้` — Claude จะแก้ในรอบถัดไป

## ความหมายของสถานะ

| สถานะ | ความหมาย |
| --- | --- |
| รอทำ | สั่งแล้ว รอ Claude ทำ |
| กำลังทำ | Claude กำลังทำอยู่ |
| เสร็จ — รอตรวจ | ทำเสร็จแล้ว รอเจ้าของดู |
| สั่งแก้ | เจ้าของขอแก้ Claude จะทำรอบถัดไป |
| อนุมัติแล้ว | จบงาน ใช้ได้เลย |

## คำถามพบบ่อย

**Q: สั่งจากมือถือได้ไหม?** ได้ — สร้างโน้ตใน Inbox จาก Obsidian มือถือ พอ sync ถึงคอม Claude ก็เห็น
**Q: งานเก่าหายไปไหน?** งานเสร็จเกิน 30 วัน Claude ย้ายไป `Archive/` ให้อัตโนมัติ
**Q: สั่งงานที่ไม่มีใน template ได้ไหม?** ได้ ใช้ [[Templates/สั่งงาน — ปรึกษาธุรกิจ|template ปรึกษาธุรกิจ]] เขียนอิสระได้เลย
